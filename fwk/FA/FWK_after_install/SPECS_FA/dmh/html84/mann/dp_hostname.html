dp_hostname Tcl Command

NAME

dp_hostname - returns the TCP/IP hostname

SYNOPSIS

dmh::dp_hostname

DESCRIPTION

The dp_hostname command returns the TCP/IP hostname of the computer that the interpreter is running on. The returned hostname will not include the domain name. In contrast, the info hostname command will sometimes return the hostname and the domain name, as in rex.hume.com, depending on the platform and configuration.

SEE ALSO
