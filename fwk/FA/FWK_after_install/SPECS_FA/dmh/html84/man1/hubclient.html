DMH Datahub Client Application

NAME

hubclient - User Interface Client for the Datahub

SYNOPSIS

hubclient ?mailbox@groupname?

hub84client ?mailbox@groupname?

DESCRIPTION

The hub84client application script is used to provide a user interface for a Hume Datahub that is executing remotely, or in the background. This script is also installed under the generic name hubclient if you do not have a prior version of the Hume Datahub SDK installed. The script uses the DMH message system capabilities to connect to the target application which can be anywhere your TCP/IP software can reach.

The optional command line argument can specify the command mailbox and DMH message system [hostname:]groupname of the target Datahub. If the command line argument is not provided, the application script prompts the user for it.
