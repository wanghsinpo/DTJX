Tk Widget Demo Application

NAME

widget - Tk Widget Demo Application

SYNOPSIS

cd /usr/local/lib/tk8.4/demos; dmh_wish -f widget

DESCRIPTION

The installation of Tcl/Tk usually includes the installation of demonstration scripts in the subdirectory demos of the Tk library directory. These scripts are very useful to see capabilities of the Tk windowing user interface software, and example code. The script named widget is the usual one to execute from the command line, or to source from a wish shell prompt.
